Сервер: 
+ Вывод внешнего ip-адреса на панели
+ Мониторинг подключения клиентов и вывод их адресов

Клиент:
+ Ручной ввод ip сервера и никнейма
+ Оповещение об успешном соединении
+ Стилизованные шрифты и цвет фона окна
+ Обработчик нажатия enter при отправке сообщения
+ Оповещение о сообщении при неактивном окне чата
+ Кнопка exit для выхода из клиента и отправке соответствующего сообщения на сервер и в чат
